About Kurd Spell

The research project behind the publications.

Research Background

Kurdish is an under-resourced language with limited software tooling. Kurd Spell addresses this gap by providing a standardized spell-checking engine with Kurdish-specific phonological and morphological rules.

The library is written in .NET and exposes a clean API for integration into text editors, web apps, and developer pipelines.

Technical Details

  • Language: .NET (C#)
  • Approach: Rule-based NLP + dictionary lookup
  • Supports: Sorani Kurdish script
  • Output: Ranked spelling suggestions
  • Integration: Standard library API
