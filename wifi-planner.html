Project Case Study

Wi‑Fi Planner — Floor Simulation Multi‑Vendor RSSI Calculator

A real‑time Wi‑Fi RSSI calculator for network planning. The tool estimates signal strength based on AP power, frequency, distance, and wall/floor obstacles. It runs fully in the browser and supports multiple vendors.

Tech

Mode

Use Case

Wi‑Fi Planning

Core Value

Instant RSSI feedback with multi‑vendor flexibility and offline‑friendly usage.

RSSI Path Loss Multi‑AP

Key Features

Practical capabilities designed for quick and accurate planning.

Dynamic RSSI Calculation

Updates instantly when AP power, frequency, distance, or wall types change.

Wall & Floor Modeling

Concrete (15 dB), plastic (5 dB), and glass (7 dB) attenuation profiles.

Summary View

Totals walls/floors, overall loss, and color‑coded RSSI health indicators.

Lightweight UI

Clean Bootstrap 5 interface that runs fully in the browser with no backend.

Screenshot

Real interface preview.

Algorithms & Logic

Transparent calculations for predictable planning.

Distance‑Loss Formula

Log‑Distance Path Loss (LDPL) — simplified for practical use.

RSSI Estimation

Free‑Space Path Loss approximation for signal prediction.

Multi‑AP Selection

Chooses the strongest signal (max RSSI) for each zone.

Device Suggestion

Heuristic matching based on TX power and capacity tolerance.
